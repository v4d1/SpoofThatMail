# SpoofThatMail
Bash script to check if a domain or list of domains can be spoofed based in DMARC records


File with domains (One per line):
```
sh SpoofThatMail.sh -f domains.txt
```
One single domain:
```
sh SpoofThatMail.sh -d domain
```
